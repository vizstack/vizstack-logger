vz-logger

A web-based visual logger
for debugging.

Visualize data structures from programs in different languages.


Get Started

Interactive Visuals

Takes print statements to the next level by showing data structures as interactive, composable visualizations, built with Vizstack.

Search and Filter

Stop scrolling through long lists of text! Use tags, filenames, and levels to easily find the messages you're looking for.

Multiple Languages

Different processes, written in different languages, can all log to the same place. Makes distributed debugging much less painful!
